# Product Scope

## Product

`alsu-me` is a personal website for Alsu Karimova, presenting tutoring, translation, and interpreting services.

## Primary Surfaces

- Hero section with brand introduction and call to action
- About section with credentials and personal presentation
- Services section for tutor and translator/interpreter offerings
- Prices section
- Testimonials section
- Contacts section
- Footer and navigation

## Current Boundaries

- The app is a static frontend experience.
- There is no backend, authentication, database, routing layer, or checkout flow.
- Vercel Analytics is the only analytics integration currently visible in app code.

## Implementation Bias

- Preserve the single-page flow unless a task explicitly requires routing.
- Keep content and visuals inspectable and asset-driven.
- Avoid broad framework or design-system changes for naming or styling tasks.
